Маршрутизаторы Cisco 1000 Series
Cisco 1000 - это серия компактных, простых в установке и удобных в эксплуатации настольных машрутизаторов для удаленных офисов.

Маршрутизаторы Cisco 1000 обеспечивают, недорогие, высокопроизводительные соединения по синхронным, асинхронным линиям или каналу ISDN. IP-маршрутизация поддерживает экономичный доступ в сеть Internet для таких приложений, как электронная почта, передача файлов, просмотр страниц World Wide Web. Маршрутизация протокола IPX поддерживает популярные службы сетей Novell NetWare. Для сетей с компьютерами Macintosh поддерживается маршрутизация AppleTalk. 

Основные функции и преимущества: 

Поддержка маршрутизации по требованию (dial-on-demand routing - DDR) 
Поддержка функции IPX Protocol Spoofing 
Поддержка загрузки программного обеспечения с TFTP сервера 
Flash ROM позволяет сохранять файлы операционной системы и конфигурационные файлы и загружать их при запуске маршрутизатора 
Поддержка маршрутизации между сетями по выделенным линиям, по сетям Frame Relay, SMDS, X.25 
Маршрутизация протоколов IP, IPX, AppleTalk 
Поддержка функций прозрачного моста по алгоритму остовного дерева (Spanning Tree) 
Поддержка WAN-протоколов: HDLC, LAPB, PPP 
Управление по протоколам SNMP, Telnet или с помощью консольного порта 
Cisco 1000 - это серия компактных, простых в установке и использовании настольных машрутизаторов, которые предлагают экономичные по стоимости, высокопроизводительные соединения по синхронным, асинхронным линиям или ISDN. IP-маршрутизация обеспечивает экономичный доступ в сеть Internet для таких приложений как электронная почта, передача файлов, просмотр страниц World Wide Web. Маршрутизация протокола IPX поддерживает популярные службы сетей Novell NetWare. Для сетей с компьютерами Macintosh поддерживается маршрутизация AppleTalk. Каждый из маршрутизаторов серии Cisco 1000 поддерживает режим "Dial-on-demand" и целый ряд других свойств, позволяющих снизить стоимость и увеличить эффективность WAN соединений.

Серия состоит из 2-х моделей: 
Cisco 1003 - оснащен 1 портом Ethernet, 1 портом ISDN BRI 
Cisco 1005 - оснащён 1 портом Ethernet, 1 универсальным последовательным Sync/Async портом 
Наиболее популярной, из за своей универсальности, стала модель Cisco 1005. 

Модель Cisco 1005 
Один порт Ethernet 10BASE-T с разъёмом RJ45 
Один последовательный WAN порт с 60-ти контактным разъемом. Тип порта определяется подключённым кабелем. В синхронном режиме порт может работать на скоростях до 2Мб/с, а в асинхронном режиме(EIA/TIA-232) до 115.2 Кб/с. Поддержиаются интерфейсы EIA/TIA-232, X.21, V.35, EIA/TIA-449, EIA-530. 
Консольный порт с разъёмом RJ-45 
Один слот PCMCIA Type 2 для Flash ROM.

Сфера применения

Cisco 1005 позволяет просто и экономично организовать доступ удалённых филиалов в корпоративную сеть головной организации, или обьединить сети разнесенных по городу подразделений. 

Эта модель сможет обеспечить гибкий и эффективный доступ в сеть Internet.
